# Todo Push

Use `../scripts/push <project-name> <title> -d <plan>` with the `agents`
Mamba environment to add a finalized plan to the named project's queue.

The project name must be explicit. Keep the title and plan as separate shell
arguments, and do not push an incomplete plan.
